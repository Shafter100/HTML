1. Вынести css в отдельный файл
2. Расскидать все по папкам
3. Подключить шрифт, можно из https://fonts.google.com
4. Посмотри как выглядит в сафари
5. Добавь кодировку utf-8
6. Лучше не писать селекторы по тэгу и id
7. Для разметки лучше использовать новые тэги header, section... Поисковикам и скринридерам будет проще понять, что находится на твоей странице
8. Для сброса браузерных стилей подключи какой-нибудь normalize.css
9. У кнопки назад не менятеся цвет иконки, кнопка должна быть button или link. Иконку лучше добавить через ::before
10. Тэг strong делает текст важным, а не жирным
11. Плохо использовать инлайновые стили в тэгах, их тяжело перекрыть, лучше использовать селектор с более высоким приоритетом
12. Позиционирование внутрених элементов нужно делать отностительно родителя, а не общего блока. Это про Гришу и скидку в тарифе целый год.
13. Слишком много id, на этой странице они нужно только для чекбоксов. Используй id только когда это необходимо.
14. Для ссылок и кнопок можно испльзовать transition, для красоты, у нас он обычно .3s
